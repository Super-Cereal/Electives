Факультативы - проект, построенный на python с использованием flask

Весь исходный код можно посмотреть по ссылке, и все используемые модули можно увидеть в файле requirements.txt

Все возможности и особенности этого проекта


  • Регистрация и авторизация пользователей + Разделение страниц сайта на приватные и общедоступные
  • Зарегистрированный пользователь может управлять данными своего профиля, менять изображения на главной странице своего профиля
  • Зарегистрированные пользователи делятся на три типа:
    1. Ученик - в самом низу пирамиды, он не может создать факультатив или быть назначен руководителем
    2. Учитель - может создать факультатив под своим руководством
    3. Администрация - может создать факультатив под руководством любого пользователя кроме ученика
  • Руководитель факультатива может:
    1. Удалить свой факультатив
    2. Редактировать его и изменять его изображение
    3. Создавать, редактировать и удалять задания для участников факультатива, прикрепить любой файл к заданию
  • Реализовано Api с минимальной общедоступной функциональностью:
    1. Получить информацию о пользователе/пользователях, зарегистрироваться
    2. Получить информацию о факультативе/факультативах
  • Можно отсортировать и отфильтровать все данные находящиеся на странице под свои нужды
  • Базовая бизнес-логика:
    1. Ведется подсчет участников каждого факультатива
    2. При удалении пользователя все факультативы под его руководством самоликвидируются, а папки с их файлами автоматически очищаются
    3. Вышестоящие типы пользователей имеют больше возможностей, так, администрация может удалить любой факультатив и любого пользователя; однако она не может редактировать чужие данные
    4. При вступлении в факультатив все задания, которые он предоставляет, отображаются в отдельной вкладке "Мои задания"
  • Большое колличество связанных сущностей, всего сущностей - 4, общее число связей - 5
  • Есть сущность, способная управлять любыми типами файлов и изображений; каждая прочая сущность с ней связана
  • У большинства сущностей есть отдельный файл с обработчиками адресов, так что поиск и исправление багов и недочетов осуществляется очень быстро
  • Сайт защищен от csrf-аттак, различных иньекций, а так же манипуляций с именами подгружаемых файлов
  • Из-за особенности системы безопасности, все файлы, подгружаемые на сайт, должны иметь английское название
    Файлы, подгружаемые на сайт могут иметь любое название
